
# Projeto UNES – Site Institucional

Um site estático e simples para uma universidade fictícia (UNES), criado para praticar conceitos fundamentais de **HTML** (estrutura de páginas, links, imagens, tabelas e formulários).  
Ideal como primeiro projeto de marcação, sem dependências externas.

## ✨ Visão geral

- **Tipo**: Site estático (HTML puro)  
- **Páginas**: `index.html` (Home), `quem-somos.html` (Sobre), `contato.html` (Contato)  
- **Objetivo**: Exercitar a estruturação de páginas e navegação básica


---

## 🖼️ Preview

- **Home**: Apresentação da instituição, banner/capa e links de navegação.
- **Quem Somos**: Texto institucional, missão/visão/valores e/ou lista de motivos para estudar na UNES.
- **Contato**: Formulário simples (nome, e‑mail, assunto/mensagem) e/ou informações de endereço.
